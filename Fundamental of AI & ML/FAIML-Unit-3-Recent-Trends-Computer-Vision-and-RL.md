## 6. Reinforcement Learning (RL)

---

### Definition

Reinforcement Learning is a type of ML where an agent learns by interacting with environment and maximizing rewards.

---

### Core Components

* Agent → decision maker
* Environment → where agent acts
* Action → possible moves
* State → current situation
* Reward → feedback

---

### Working Process

1. Agent observes state
2. Takes action
3. Environment changes state
4. Reward is given
5. Agent learns and improves

---

### Goal:

Maximize cumulative reward

---

### Real World Applications

* Game playing (AlphaGo)
* Robotics
* Resource management
